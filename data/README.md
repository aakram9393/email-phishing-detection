Dataset name: Enron Spam Data

Source (public): Publicly available Enron email dataset released for research purposes.

What it contains: The Enron dataset contains real corporate emails collected from the Enron
Corporation. It includes both legitimate emails (ham) and spam/phishing
emails, making it suitable for email classification tasks.


Labels: - ham: legitimate email
- spam: phishing or unsolicited email


Data usage: - Raw data will be stored unchanged
- Processed data will be generated via reproducible scripts
- No manual label modification will be performed



Known limitations: - Dataset contains older email language
- May not reflect modern phishing techniques
- Class imbalance may exist
